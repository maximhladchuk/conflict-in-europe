Rules

I. Discord Rules

Forbidden:

  • 1.1 Degrading the honor and dignity of other server participants in any way
  • 1.2 Mentioning close relatives in a negative context
  • 1.3 Distribution of third-party resources and servers in our Discord or in the game
  • 1.4 Discussing or mentioning real-life wars, conflicts, or events that may cause tension within the server community
  • 1.5 Political clashes
  • 1.6 Playing music or extraneous sounds in voice channels
  • 1.7 Using programs to change your voice
  • 1.8 Condemning the actions of the server administration in the general channels (allowed in the appropriate channel: appealing-a-ban)
  • 1.9 Misleading the project administration
  • 1.10 Making comments in a complaint to which you have no direct relation, except when providing additional evidence
II. Game Server Rules

Forbidden:

  • 2.1 Using any third-party programs that give an advantage in the game
  • 2.2 Mentioning close relatives of the player in a negative context
  • 2.3 Do not cross the red line near the enemy's main base
  • 2.4 Deliberately disrupting the game of other players in any way
  • 2.4-1 The use of artillery, MLRS, mortars, and command vehicles is prohibited unless the red line has been crossed
  • 2.5 Using bugs that give you an advantage over other players
  • 2.6 Personal insults in general chat unrelated to game immersion. Personal insults are only allowed in voice chat against enemies
  • 2.7 Engaging in political discussions in the game and mentioning real-world war events is prohibited
  • 2.8 Any insults to the administration wherever it may be
